Tips2 - Reade more
主题取消了繁琐的翻页模式，仅保留首页文章列表，其他文章均导向到文章归档页面，方便查看所有文章，是我认为比翻页更好的体验方式。

新建页面，选择页面模板位“文章归档”，固定连接设置为“archives”！
